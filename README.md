# Numero Secreto 🎯

Projeto desenvolvido com o objetivo de praticar conceitos de lógica de programação utilizando HTML, CSS e JavaScript.

## Sobre o projeto 📌

Este projeto foi criado para estudo e prática de lógica de programação, com o objetivo de aplicar conceitos fundamentais, como:

- variáveis

- condicionais

- funções

- interação com o usuário

- manipulação do DOM

O jogo consiste em adivinhar um número secreto gerado aleatoriamente pelo sistema.

## Tecnologias 🤖

- [X] HTML
- [X] CSS
- [X] JavaScript 

## Como executar o projeto 📁

1- Clone o repositório:

- git clone https://github.com/LarissaQuintes/projeto-numero-secreto

2- Abra a pasta no VS Code.

3- Execute o arquivo index.html no navegador
ou utilize a extensão Live Server.

## Como jogar 🎮

1 -  Digite um número no campo indicado.

2 - Clique no botão para verificar.

3 - O sistema informará se o número secreto é:

- 🔼 Maior

- 🔽 Menor

4 - Continue tentando até acertar.

5 - Após acertar, é possível iniciar um novo jogo

## Funcionalidades 💡

- Geração aleatória do número secreto

- Validação das tentativas

- Dicas ao jogador

- Reinício do jogo

- Interface simples e interativa
